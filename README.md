# The Data Hub

A RESTful API server built with Node.js and Express that demonstrates backend development fundamentals, including route handling, CRUD operations, JSON request processing, and API testing with Postman.

## Features
- Express.js server running on Port 5000
- RESTful API architecture
- Create, Read, Update, and Delete (CRUD) operations
- In-memory data storage using JavaScript arrays
- JSON request and response handling
- API testing with Postman
- Beginner-friendly backend project

## Tech Stack
- Node.js
- Express.js
- Postman
- JavaScript
